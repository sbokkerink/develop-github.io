JetBrains, initially called IntelliJ Software, was founded in 2000 in Prague by three Russian software developers: Sergey Dmitriev, Valentin Kipyatkov and Eugene Belyaev. The company's first product was IntelliJ Renamer, a tool for code refactoring in Java. In 2012 CEO Sergey Dmitriev left the company and was replaced by Oleg Stepanov and Maxim Shafirov. In 2021 The New York Times stated that unknown parties might have embedded malware in JetBrains' software that led to the SolarWinds hack and other widespread security compromises. In a press release, JetBrains said they had not been contacted by any government or security agency and had not "taken part or been involved in this attack in any way". The CEO of one of the affected companies, SolarWinds, "asked about the possibility that software tools made by JetBrains, which speeds the development and testing of code, was the pathway, Mr. Ramakrishna said there was still no evidence". In response to the 2022 Russian invasion of Ukraine, the company suspended sales and R&D activities in Russia indefinitely as well as sales in Belarus.

Visual Studio Code is a source code editor developed by Microsoft for Windows, Linux, and macOS. It includes support for debugging, built-in Git checking, syntax highlighting, intelligent code completion, code snippets, and code refactoring. It is also customizable, allowing users to change the editor theme, keyboard shortcuts, and preferences. It is free and open source, although the official download is licensed. Visual Studio Code is based on Electron, a framework used to implement Node.js applications for the desktop and running on the Blink layout engine. While it uses the Electron framework, it does not use Atom and instead uses the same editor component used in Visual Studio Team Services. Visual Studio Code was announced by Microsoft on April 29, 2015 at the 2015 Build conference. Shortly after, a preview build was released. On November 18, 2015, Visual Studio Code was released under the MIT license and the source code was posted on GitHub. Support for extensions was also announced. On April 14, 2016, Visual Studio Code exited the preview build phase.
